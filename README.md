# Glamour Gate - The Fashion Landmark in Sri Lanka

![Glamour Gate Logo](public/assets/logo.png)

## 🎯 About The Project

Glamour Gate is a premium customer registration and identity verification system for the leading fashion landmark in Negombo, Sri Lanka. This application provides secure OTP-based phone verification and Sri Lankan NIC validation for customer onboarding.

## ✨ Features

- 📱 **OTP Verification** - Secure phone number verification
- 🆔 **Sri Lankan NIC Validation** - Supports both old (9 digits + V/X) and new (12 digits) NIC formats
- 👤 **Auto Gender Detection** - Automatically detects gender from NIC number
- 🎂 **Birthday Validation** - Cross-validates birthday with NIC data
- 🎨 **Modern UI** - Glassmorphism design with smooth animations
- 📱 **Fully Responsive** - Works on all devices
- 🌙 **Dark Theme** - Premium dark blue color scheme

## 🚀 Tech Stack

- **Frontend:** React 18, Tailwind CSS
- **HTTP Client:** Axios
- **Icons:** Font Awesome 6, Heroicons
- **Animations:** Custom CSS animations

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager
